# Voice chat

Voice chat is built in in-game. Simply [setup your voice chat](#how-to-use-voice-chat-in-game) & start talking. 

## How to use voice chat in game? 
1. Install Fabric launcher http://fabricmc.net/use/installer/
2. Install Simple Voice Chat https://www.curseforge.com/minecraft/mc-mods/simple-voice-chat
